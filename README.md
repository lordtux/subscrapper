# subscrapper
An automatic way to manage subtitules

Script mode:
```sh
$ python sub-scrapper.py -m script -q <query text> -l <result limit>
```

Scanner mode:
```sh
$ python sub-scrapper.py -m scanner
```

Server mode (work in progress):
```sh
$ python sub-scrapper.py -m server
```
